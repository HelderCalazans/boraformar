Gerar Simulado Aleatório

Bem Vindo ao gerador de provas

Aqui você poderá criar seus simulados seguindo as seguintes regras:

1 - Cada simulado poderá ter até 10 questões.

2 - Cada simulado poderá ter até 3 assuntos.

3 - Você pode definir quantas questões de cada assunto deseja, respeitando as regras 1 e 2.

Após criar o simulado, poderá utilizar o código gerado para compartilhar sua prova com seus colegas.
